    • Defined in apis/EpisodesApi.ts:31

    Get an Episode

    Get Spotify catalog information for a single episode identified by its unique Spotify ID.

    Required Scopes: user-read-playback-position for reading the user's resume points.

    example
    const episode = await spotify.episodes.getEpisode('5VhaAEtTrKQ1NSneHsXK6C');
console.log(episode.name);
// "S1E6 - Institutionalized by Kendrick Lamar"

getEpisodes

    • Defined in apis/EpisodesApi.ts:60

    Get Several Episodes

    Get Spotify catalog information for multiple episodes based on their Spotify IDs.

    Required Scopes: user-read-playback-position for reading the user's resume points.

    example
    const episodes = await spotify.episodes.getEpisodes([
 '3Ht9rzglNNgJBFGp2RL7o2',
 '7r40piSdCht1NL720dKNWf',
]);
console.log(episodes.map(episode => episode.name));
// Array [ "The Soul Train Episode", "How the Voyager Golden Records Work" ]

    Parameters

    • episodeIds: string[]

      A list of the Spotify IDs for the episodes.

    • Optional options: MarketOptions

      Optional request information.

    Returns Promise<Array<Episode | null>>
